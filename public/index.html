⌨️ Commandes du bot
🧪 Session de test
!teststart Démarre une session de test — le bot répond en mode isolé pour tester le comportement.
!testnew Réinitialise la session de test en cours sans l'arrêter.
!teststop Arrête la session de test en cours et revient au mode normal.
⚙️ Configuration & données
!reloadconfig Recharge la configuration depuis JSONbin sans redémarrer le bot.
!saveprofiles Force la sauvegarde immédiate des profils membres dans JSONbin.
👤 Profils & mémoire
!memoire Affiche ce que le bot se souvient de toi — profil, niveau d'amitié, incidents.
!profil @membre Affiche le profil complet d'un membre spécifique (remplace @membre par la mention).
🔧 Divers
!test Vérifie que le bot est en ligne et répond correctement.
